Examples

以下代码启用了消息缓存，并配置客户端在房间可用时始终下载用户。

var config = new HeyBoxSocketConfig
{
    AlwaysDownloadUsers = true,
    MessageCacheSize = 100
};
var client = new HeyBoxSocketClient(config);

Remarks

此配置基于 HeyBoxRestConfig，在与 REST 有关的配置的基础上，定义了有关网关的配置。
